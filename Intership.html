INTERSHIP

AZURE BACKEND DEVELOPER

AUGUST 2023 - SEPTEMBER 2023

The project began with the creation of HTTP-triggered functions, implementing both GET and POST methods, and ensuring proper authorization for secure access. enhancements were made to establish connections with Azure Storage Accounts. This integration enabled the API to interact with cloud storage, facilitating scalable and persistent data storage solutions A noteworthy aspect of the project was the integration of Swagger UI, providing a user-friendly interface accessible through a URL. In AZURE resource groups were created , Azure Storage Accounts provide a scalable and secure cloud storage solution within the Microsoft Azure ecosystem , Azure Function App is a serverless computing service offered by Microsoft Azure, providing a platform for building, deploying, and running event-triggered functions.
